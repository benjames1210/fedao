## fedao

![](https://img.shields.io/npm/v/fedao.svg) ![](https://img.shields.io/npm/dm/fedao.svg)

**fedao — 学而思在线前端工程构建工具**

### 主要特征

- 基于fis3的前端工程化解决方案
- 全面支持es6/es7
- 支持npm生态
- 支持智能打包
- 内置 `autoprefixer`、`px2rem` 等实用插件
- 数据mock
- 远程部署
- 代码区块裁剪

### 安装

```
npm install -g fedao
```

### 主要命令

```
fedo server start|stop|restart|open
fedao release -cw //发布
```
